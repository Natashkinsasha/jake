# Jake — Фичи приложения

Jake — AI-тьютор для изучения английского языка. Ученик общается голосом с виртуальным преподавателем "Jake", который адаптируется под уровень, интересы и слабые места ученика.

---

## 1. Авторизация

Вход через Google. При первом входе автоматически создаётся профиль и назначается тьютор. Токен сохраняется в localStorage и используется для всех запросов.

Все страницы кроме `/login` защищены — без авторизации редирект на логин.

---

## 2. Live-урок с AI-тьютором (ядро)

Fullscreen-экран разговора с тьютором в реальном времени.

### Как работает разговор

Ученик говорит в микрофон. Речь стримится через WebSocket в Deepgram (STT), который возвращает текст по сегментам. Когда ученик замолкает на 1 секунду, накопленный текст отправляется на бэкенд.

Бэкенд собирает контекст для LLM:
- Профиль ученика (имя, уровень, номер урока)
- Настройки (стиль коррекции, скорость, родной язык)
- Персональные факты из прошлых уроков (хобби, работа, семья и т.д.)
- Эмоциональный тон последних уроков
- Слабые и сильные грамматические темы
- Недавний словарь

LLM генерирует ответ от лица "Jake" — австралийского тьютора с неформальным стилем. Текст озвучивается через ElevenLabs (TTS) и отправляется клиенту как base64-аудио.

На клиенте текст появляется с анимацией (слово за словом), параллельно воспроизводится аудио.

### Упражнения в разговоре

Тьютор может вставлять упражнения прямо в поток разговора. LLM генерирует `<exercise>JSON</exercise>` теги, которые парсятся и рендерятся как интерактивные компоненты:

- **Fill the gap** — предложение с пропуском, ученик вводит слово ("I ___ a student")
- **Multiple choice** — вопрос с вариантами ответа (A, B, C, D)
- **Sentence builder** — пул слов, нужно собрать предложение в правильном порядке
- **Error correction** — предложение с ошибкой, нужно исправить

После ответа бэкенд даёт обратную связь.

### UI урока

- **Чат** — история сообщений ученика и тьютора, новые сообщения появляются с анимацией
- **Thinking-индикатор** — анимированные точки пока AI думает
- **Микрофон** — большая кнопка: зелёная = слушает, жёлтая = выключен
- **Пауза** — ставит урок на паузу
- **Таймер** — время с начала урока (MM:SS)
- **Live-транскрипт** — пока ученик говорит, текст появляется в реальном времени
- **Tab focus** — аудио ставится на паузу если ученик ушёл с вкладки

### Умная очередь сообщений

Если ученик начал говорить новое сообщение пока AI ещё отвечает на предыдущее — устаревший ответ отбрасывается, чтобы не было путаницы.

### Отказоустойчивость

Если TTS (ElevenLabs) упал — урок продолжается текстом без аудио.

## 3. Пост-обработка урока

Когда ученик нажимает "Завершить урок", запускается цепочка фоновых задач:

1. **Суммаризация** — LLM анализирует весь разговор и создаёт саммари: темы, новые слова, найденные ошибки
2. **Оценка уровня** — обновляется текущий уровень ученика (A1-C2)
3. **Словарь** — новые слова из урока добавляются в словарь со стартовым strength
4. **Грамматика** — обновляются счётчики ошибок/успехов по грамматическим темам
5. **Эмоциональный контекст** — сохраняется вектор эмоционального тона урока (для будущих уроков)
6. **Генерация ДЗ** — создаётся домашнее задание на основе материала урока

---

## 4. Память о ученике

Система извлекает и запоминает персональные факты из каждого урока. Если ученик рассказал что работает программистом, любит футбол или недавно ездил в Италию — это сохраняется по категориям (личное, работа, хобби, семья, путешествия, образование).

На следующих уроках Jake использует эти факты для построения диалога — спрашивает как дела на работе, вспоминает про поездку и т.д.

Также сохраняется "эмоциональный тон" каждого урока как vector embedding. Это позволяет Jake подстраивать стиль общения — если ученик был расстроен на прошлом уроке, Jake будет мягче.

---

## 5. Домашние задания

После каждого урока автоматически генерируется 5-7 упражнений, привязанных к материалу урока:
- Темы, которые обсуждались
- Ошибки, которые ученик допустил
- Новые слова
- Уровень ученика
- Интересы ученика (чтобы примеры были релевантные)

Типы упражнений те же что и в уроке: fill the gap, multiple choice, sentence builder, error correction.

Ученик открывает ДЗ, заполняет ответы, отправляет — получает оценку. Проверка автоматическая (case-insensitive matching).

На странице `/homework` видно все задания: дата, количество упражнений, статус выполнения, оценка.

---

## 6. Словарь и повторение

Все новые слова с уроков попадают в словарь ученика. Каждое слово имеет `strength` (0-100) — насколько хорошо ученик его знает.

### Spaced repetition

Слова предлагаются к повторению по алгоритму интервального повторения:
- Новое слово → повторение через 1 день
- Запомнил → через 3 дня, потом 7, 14, 30, 60
- Забыл → strength падает (-20), интервал сбрасывается

### Режим повторения

Карточки: показывается слово → ученик пытается вспомнить → нажимает "Показать ответ" → отмечает "Помню" или "Забыл". В конце — экран результатов: X из Y слов запомнено.

---

## 7. Прогресс и статистика

Страница `/progress` показывает:
- **Текущий уровень** — оценка A1-C2, обновляется после каждого урока
- **Грамматические темы** — по каждой теме виден уровень (0-100) и количество ошибок
- **Общая статистика** — количество пройденных уроков, выученных слов

Уровень грамматики по темам обновляется автоматически: ошибки в уроке снижают уровень, правильное использование — повышает. Это влияет на то, какие упражнения Jake предлагает.

---

## 8. Настройки ученика

На странице `/settings` ученик может настроить под себя:

- **Стиль коррекции ошибок:**
  - `immediate` — Jake поправляет сразу
  - `end_of_lesson` — ошибки разбираются в конце урока
  - `natural` — Jake корректирует через переспрашивание / natural reformulation

- **Объяснения грамматики** — вкл/выкл, хочет ли ученик подробные объяснения

- **Скорость речи** — slow / normal / fast (влияет на TTS)

- **Родной язык** — использовать ли русский для объяснений

---

## 9. Тьюторы

Система поддерживает несколько тьюторов с разными личностями, голосами и стилями. Каждый тьютор имеет:
- Имя и аватар
- Описание личности и стиль общения
- Свой системный промпт для LLM
- Свой голос в ElevenLabs
- Акцент и черты характера

Сейчас основной тьютор — **Jake** (австралиец, неформальный, использует "mate", "reckon").

## 10. Dashboard

Главная страница после входа:
- Большая кнопка "Начать урок"
- Список последних уроков: номер, дата, длительность, тема, саммари

## 11. История уроков

На странице `/lessons/:id` можно просмотреть:

- Саммари урока
- Темы, которые обсуждались
- Новые слова
- Найденные ошибки

## Статус реализации на клиенте

Все фичи полностью реализованы на бэкенде. На клиенте код тоже написан для всех фичей, но большинство **скрыты от пользователя** — нет ссылок в навигации.

| Фича | Бэкенд | Клиент: код | Клиент: доступна из UI |
|------|--------|-------------|------------------------|
| Авторизация (Google) | ✅ | ✅ | ✅ `/login` |
| Dashboard | ✅ | ✅ | ✅ Главная после входа |
| Live-урок | ✅ | ✅ | ✅ Кнопка на dashboard |
| Детали урока | ✅ | ✅ | ✅ Клик по уроку на dashboard |
| Домашние задания | ✅ | ✅ | ❌ Нет ссылки в навигации |
| Словарь | ✅ | ✅ | ❌ Нет ссылки в навигации |
| Повторение слов | ✅ | ✅ | ⚠️ Только со страницы словаря |
| Прогресс | ✅ | ✅ | ❌ Нет ссылки в навигации |
| Настройки | ✅ | ✅ | ❌ Нет ссылки в навигации |
| Тьюторы (выбор) | ✅ | ❌ | ❌ Нет страницы |

**Проблема:** `AppHeader` показывает только лого + имя пользователя + sign out. Навигации к homework, vocabulary, progress, settings нигде нет. Страницы полностью рабочие, но доступны только по прямому URL.

## Как всё связано

```
Ученик входит (Google) → Dashboard → Начать урок
                                         ↓
                              WebSocket подключение
                                         ↓
                    Микрофон → Deepgram STT → текст
                                         ↓
                    Контекст (память + прогресс + настройки)
                                         ↓
                              LLM → ответ Jake
                              (+ опциональное упражнение)
                                         ↓
                          ElevenLabs TTS → аудио
                                         ↓
                    Клиент: анимация текста + аудио + упражнение
                                         ↓
                              ← цикл повторяется →
                                         ↓
                              Завершение урока
                                         ↓
              ┌──────────────────────────────────────────┐
              │           Пост-обработка (фон)           │
              ├──────────┬──────────┬──────────┬─────────┤
              │ Саммари  │ Словарь  │ Грамм.   │   ДЗ    │
              │ + уровень│ + слова  │ прогресс │ 5-7 упр.│
              └──────────┴──────────┴──────────┴─────────┘
                                         ↓
                    Ученик: ДЗ, повторение слов, прогресс
```
